Configurar seu GESTUUM

Vamos configurar seu GESTUUM em poucos minutos. Tudo acontece aqui no navegador, sem complicação nenhuma.

Antes de começar, separe:

  • ✓ Navegador Google Chrome (computador ou Android)
  • ✓ Cabo USB-C de dados (veja o aviso abaixo)
  • ✓ Seu dispositivo GESTUUM em mãos
⚠ Atenção ao cabo USB-C!
Nem todo cabo USB-C funciona. Existem dois tipos:
✅
Cabo de DADOS
Transfere arquivos e carrega. Funciona com o GESTUUM.
❌
Cabo só de CARGA
Só carrega bateria. NÃO funciona com o GESTUUM.
Como saber? Se você já usou o cabo para transferir fotos do celular para o computador, é um cabo de dados. Cabos que vieram com carregadores baratos geralmente são só de carga. Na dúvida, use o cabo que veio com o seu celular (Samsung, Motorola, etc.).
💻 Onde funciona
✓ Windows 10+ com Google Chrome
✓ macOS 12+ com Google Chrome
✓ Linux com Google Chrome
✓ Chromebook
✓ Android 9+ com Chrome (precisa adaptador USB-C OTG)
✗ iPhone / iPad — Safari não suporta conexão USB
✗ Firefox — nenhuma plataforma

Conecte e configure

Conecte o dispositivo ao computador usando o cabo USB-C.

Quando estiver conectado, clique no botão abaixo.

Antes de conectar:

📎 Use um cabo USB-C de dados (cabos só de carga NÃO funcionam — veja dica no passo 1)
🔌 Conecte o cabo na lateral do sensor (a entrada pequena)
💻 Conecte a outra ponta em qualquer entrada USB do computador

Tranquilo: O navegador vai pedir permissão para acessar o dispositivo USB. Clique em "Permitir" — isso é necessário para a configuração e completamente seguro.
